### Color Usage Rules

- **Primary red**: Headers, buttons, links, key highlights, logo
- **Steel blue**: Professional elements, secondary headers, body text emphasis
- **Cream/Peach tones**: Soft backgrounds, card backgrounds
- **Neutrals**: Most of the visual space, let red pop against neutral backgrounds
- Use gradients sparingly for depth and visual interest

- **Letter spacing:** Normal for body, no modification needed

## Pattern Usage

See `imagery.md` for complete pattern guidelines. Quick reference:

### Pattern Strategy Selection

| Content Type | Strategy | Pattern Opacity |
|--------------|----------|-----------------|
| Hero/Cover images | Corner Framing | 15-25% at corners |
| Job postings/Hiring | No Pattern or Edge Accent | 10-15% max |
| Social media posts | Header/Footer Bands | 20-30% |
| Infographics | No Pattern | — |
| Text-heavy content | No Pattern | — |

### Key Rules

- **Never** cover entire background uniformly with pattern
- **Never** place pattern behind body text at >10% opacity
- **Always** maintain clear zones around all text content
- **Prefer** gradient fades for natural transitions
- For text-heavy content, use alternative brand signals (red borders, badges, gradient backgrounds)
