# INNOMIZE Imagery Guidelines

## Visual Style Overview

INNOMIZE imagery follows a cohesive visual language that emphasizes:
- **Innovation and technology** - Forward-thinking, modern
- **Human connection** - Teams working together, collaboration
- **Warmth through red tones** - Distinctive brand color application
- **Professionalism** - Clean, organized, purposeful

---

## Illustration Style

### Primary Style

All illustrations must use **red tones** as defined in the color palette:
- Primary: `#EA322A` (Innomize Red)
- Secondary: `#F76D55` (Coral)
- Light: `#FFC8B1` (Peach)

### Character Guidelines

For illustrations with human figures:
- **Remove facial expressions** (eyes, lips, nose) to unify illustrations from different sources
- Use simplified, faceless character style
- Consistent body proportions across illustrations
- Red/coral clothing and accents

### Illustration Sources

**Free sources:**
- [Freepik](https://www.freepik.com) - Search for "flat illustration" and recolor
- [Iconscout](https://iconscout.com) - Vector illustrations
- [Iconfinder](https://iconfinder.com) - Icon sets and illustrations

**Style keywords for sourcing:**
- Flat illustration
- Minimal characters
- Tech/business illustration
- Red duotone

### Recoloring Process

When adapting illustrations from external sources:
1. Convert to brand red palette
2. Remove facial features for consistency
3. Simplify if overly detailed
4. Ensure clean lines and shapes

## Photography Style

### Treatment Options

| Style | Description | Usage |
|-------|-------------|-------|
| **B&W** | Grayscale conversion | Professional, formal contexts |
| **B&W + Red Fade** | Grayscale with red gradient overlay | Hero sections, feature images |
| **Original** | Full color (when appropriate) | Candid team photos, events |

### Photo Overlay Effect

For the signature INNOMIZE photo treatment:
1. Convert photo to grayscale
2. Apply red gradient overlay (`#EA322A` at 30-50% opacity)
3. Gradient direction: typically from bottom or corner

### Subject Matter

**Preferred subjects:**
- Team collaboration and meetings
- Office environment and workspace
- Technology and development
- Problem-solving and planning
- Client interactions

**Avoid:**
- Generic stock photos with forced smiles
- Overly staged corporate imagery
- Cluttered or messy backgrounds
- Low-quality or pixelated images

### Photo Requirements

| Property | Requirement |
|----------|-------------|
| Resolution | Minimum 1200px width for web |
| Aspect ratio | 16:9, 4:3, or 1:1 depending on placement |
| Quality | Sharp, well-lit, professional |
| Background | Clean, uncluttered |

## Pattern

### Brand Pattern

The INNOMIZE pattern represents:
- **Change and adaptation** - Get out of comfort zone
- **Think outside the box** - Innovation
- **Transform** - Evolution and growth

### Pattern Elements

The pattern uses two alternating shapes:
- **Rounded square** (from logo container)
- **Circle** (representing transformation)

### Pattern Variants

| Variant | Style | Usage |
|---------|-------|-------|
| **Outline** | Thin stroke, transparent fill | Subtle backgrounds, watermarks |
| **Filled** | Solid shapes | Bold applications, covers |
| **Mixed** | Combination of outline and filled | Decorative elements |

### Pattern Colors

| Application | Colors |
|-------------|--------|
| Light background | Red outline on white |
| Dark background | White outline on charcoal |
| Brand emphasis | Solid red on cream |

### Pattern Opacity Guidelines

| Context | Max Opacity | Notes |
|---------|-------------|-------|
| Behind text content | 5-10% | Must not interfere with readability |
| Decorative corners/edges | 15-25% | Can be more visible when away from content |
| Header/footer bands | 20-30% | Contained areas without text overlap |
| Pure decorative (no text) | 40-60% | Only in image-focused areas |

---

### Pattern Placement Strategies

Choose the placement strategy based on content density and layout:

#### Strategy 1: Corner Framing
**Best for:** Hero sections, feature cards, promotional graphics

Place pattern in opposite corners (top-left + bottom-right or top-right + bottom-left) with gradient fade toward center.

```
┌─────────────────────────────┐
│ ○ ○ ○ ·                     │
│ ○ ○ · ·                     │
│ ○ · ·                       │
│ · ·         [Content]       │
│                       · · ○ │
│                     · · ○ ○ │
│                   · ○ ○ ○ ○ │
└─────────────────────────────┘
```

**Implementation:**
- Pattern opacity: 15-25% at corners, fading to 0% toward center
- Clear zone: Minimum 40% of canvas center must be pattern-free
- Fade gradient: Radial or linear, 150-200px transition

#### Strategy 2: Edge Accent
**Best for:** Text-heavy content, job postings, articles, forms

Single vertical or horizontal strip along one edge only.

```
┌─────────────────────────────┐
│ ○ │                         │
│ □ │                         │
│ ○ │    [Clean Content]      │
│ □ │                         │
│ ○ │                         │
└─────────────────────────────┘
```

**Implementation:**
- Pattern width: 60-100px strip
- Pattern opacity: 10-15%
- Content margin: Minimum 24px from pattern edge
- Position: Left edge preferred (follows reading direction)

#### Strategy 3: Header/Footer Bands
**Best for:** Social media posts, announcements, event graphics

Horizontal bands at top and/or bottom, content area completely clean.

```
┌─────────────────────────────┐
│ ○ □ ○ □ ○ □ ○ □ ○ □ ○ □ ○ │  ← 80-120px band
├─────────────────────────────┤
│                             │
│        [Content]            │
│                             │
├─────────────────────────────┤
│ ○ □ ○ □ ○ □ ○ □ ○ □ ○ □ ○ │  ← 80-120px band
└─────────────────────────────┘
```

**Implementation:**
- Band height: 80-120px
- Pattern opacity: 20-30%
- Clear separation: Use subtle border or color shift between band and content
- Can use with gradient backgrounds in content area

#### Strategy 4: No Pattern (Clean Background)
**Best for:** Dense text, job listings, detailed information, forms

Skip pattern entirely. Use other brand elements for recognition.

```
┌─────────────────────────────┐
│                             │
│  ┌─────────────────────┐    │
│  │ Red accent border   │    │
│  │ or gradient bg      │    │
│  └─────────────────────┘    │
│                             │
│        [Content]            │
│                             │
└─────────────────────────────┘
```

**Alternative brand signals:**
- Solid cream `#FFF3E2` or gradient background (white → cream)
- Red left border on cards (4px solid `#EA322A`)
- Red accent badges or icons
- Brand gradient in header section only
- Logo placement for recognition

---

### Pattern Strategy Selection Guide

| Content Type | Recommended Strategy | Reason |
|--------------|---------------------|--------|
| Hero/Cover images | Corner Framing | Visual impact without blocking focal point |
| Job postings/Hiring | No Pattern or Edge Accent | Text-heavy, readability priority |
| Social media posts | Header/Footer Bands | Clear content zone, brand framing |
| Event announcements | Corner Framing | Balance of branding and information |
| Articles/Blog posts | No Pattern | Reading focus |
| Product showcases | Edge Accent | Subtle branding, product focus |
| Infographics | No Pattern | Complex visual content needs clean space |
| Testimonials | Corner Framing | Personal content with brand frame |
| Forms/Applications | No Pattern | Functional, distraction-free |
| Team/Company intro | Edge Accent or Bands | Professional with brand identity |

---

### Pattern Usage Rules

**Do:**
- Choose strategy based on content density
- Maintain clear zones around all text
- Use gradient fades for natural transitions
- Test readability before finalizing
- Apply to corners or edges of designs
- Use in footer or header areas
- Create visual interest in large empty spaces

**Don't:**
- Cover entire background uniformly
- Place pattern behind body text at >10% opacity
- Use pattern when content is already visually complex
- Overwhelm the main content
- Use at full opacity over important text
- Stretch or distort the pattern
- Mix with other patterns
