# INNOMIZE Logo Guidelines

## Brand Identity

**Brand Name:** INNOMIZE
**Tagline:** Better Solutions - Better Lives
**Logo Type:** Brandmark + Wordmark combination


## Logo Variants

### 1. Horizontal (Primary)

The default logo option. Symbol on left, wordmark on right.

```
[inn] innomize
```

**Use for:**
- Headers and navigation
- Email signatures
- Business cards
- Standard web placements

### 2. Vertical (Stacked)

Symbol above wordmark. Use when horizontal space is limited.

```
  [inn]
innomize
```

**Use for:**
- Square formats
- App icons (with text)
- Social media profiles
- Limited width areas

### 3. Brandmark Only

The symbol/icon without wordmark.

```
[inn]
```

**Use for:**
- Favicons (16x16px)
- App icons
- Social media avatars
- Small placements where wordmark won't be legible

### 4. Symbol Only

The "inn" letterform without the container.

```
inn
```

**Use for:**
- Watermarks
- Pattern elements
- Very small sizes


## Logo Composition

### Brandmark Details

- **Shape:** Rounded square container with "inn" letterform
- **Corner radius:** Large (approximately 20% of height)
- **Symbol:** Custom "inn" letterform representing Innovation, Ideas, and Integration

### Clear Space

Minimum clear space around logo: **2x** the height of the symbol

```
     ┌─────────────────────────┐
     │                         │
  2x │  [inn] innomize         │ 2x
     │                         │
     └─────────────────────────┘
           ↑       2x      ↑
```

### Minimum Sizes

| Variant | Minimum Width | Usage |
|---------|--------------|-------|
| Horizontal | 120px | Web, print |
| Vertical | 60px | Web, print |
| Brandmark | 24px | Digital |
| Favicon | 16px | Browser tabs |

## Logo Placement

### Preferred Positions

The logo should be placed on the **left** or **center** of layouts.

**Do:**
- Top-left of documents and web pages
- Center for hero sections and covers
- Bottom-left for footers

**Don't:**
- Avoid right-aligned logo placement (due to symbol weight on left)

### Background Guidelines

| Background | Logo Version |
|------------|--------------|
| White / Light | Primary (red symbol, dark wordmark) |
| Cream / Peach | Primary (red symbol, dark wordmark) |
| Red | Reversed (white symbol, white wordmark) |
| Dark Blue / Steel | Reversed (white) |
| Charcoal / Black | Reversed (white) |
| Photos (light) | Primary with clear space |
| Photos (dark) | Reversed with clear space |


## Logo Misuse

**Never:**
- Scale disproportionately (stretch or compress)
- Change the logo colors to non-brand colors
- Rotate or reflect the logo
- Use stroke/outline only versions
- Change transparency/opacity
- Replace the font with another typeface
- Add effects (shadows, gradients, glows)
- Place on busy backgrounds without clear space
- Crop or cut off parts of the logo

## Co-Branding

When the INNOMIZE logo appears alongside partner/client logos:

1. Maintain equal visual weight
2. Use consistent spacing between logos
3. Separate with a vertical divider line if needed
4. INNOMIZE logo typically appears first (left) or in primary position
