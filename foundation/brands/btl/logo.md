### Profile Picture Guidelines

- **Shape:** Display as circle (most social platforms)
- **Minimum size:** 400x400px for quality across platforms
- **Background:** Should work on both light and dark backgrounds
- **Consistency:** Use the same profile picture across all platforms

## Placement Rules

### Profile Picture Placement

| Context | Position | Size |
|---------|----------|------|
| **LinkedIn cover** | Bottom-left or integrated with design | 80-120px |
| **Content cards** | Top-left header | 40-48px |
| **Author byline** | Inline with name | 32-40px |

### Clear Space

- Minimum clear space around profile picture: 16px
- Minimum clear space around text mark: 12px

### Do's and Don'ts

**Do:**
- Use high-resolution profile picture
- Maintain consistent sizing across similar contexts
- Pair profile picture with name for recognition

**Don't:**
- Stretch or distort the profile picture
- Use low-resolution versions
- Place on busy backgrounds that reduce visibility
- Crop the profile picture differently per platform
