Avoiding AI-Common Font Patterns

Based on ai-frontend-issue.md, we're avoiding: Inter (overused in AI starters), Open Sans (web default), Poppins (common in AI scaffolds), Geist (Vercel template smell), and default font-sans stacks. Typography is a primary brand lever - same fonts = same feel.

Final Recommendation: Option C (IBM Plex Sans + Public Sans)
